# Diabetes Prediction using Decision Tree

This project performs Exploratory Data Analysis (EDA) on the PIMA Diabetes dataset
and builds a Decision Tree model to predict diabetes outcomes.

## Dataset
- diabetes.csv

## Steps Performed
- Data cleaning & EDA
- Data visualization
- Train-test split
- Decision Tree classification
- Model evaluation

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Platform
- Google Colab
